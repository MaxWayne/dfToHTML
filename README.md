# cb
- cd into the dir where you downloaded dfToHTML
- virtualenv venv (if you don't have it, sudo pip install virtualenv)
- . venv/bin/activate
- pip install -r requirements.txt (this will take a while)
- python run.py
- go to http://0.0.0.0:8000/opt1#
